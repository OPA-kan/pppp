
```
✅ 総分析数、正解数、誤解数を表示

```

### 4️⃣ **PADC スコア解釈システム** 🧠
```
✅ P(快感度) × A(覚醒度) で感情判定
   🎉 興奮・喜び
   😊 満足・リラックス
   😰 不安・ストレス
   😢 悲しみ・落ち込み

✅ センチメント判定
   🌟 ポジティブ (P > 0.5)
   😐 ニュートラル (0.35 ≤ P ≤ 0.5)
   😔 ネガティブ (P < 0.35)

✅ 強度レベル判定
   🌤️  低い (0.0~0.5)
   ⚡ 中程度 (0.5~0.7)
   💥 非常に強い (0.7~1.0)
```

## 📊 PADC スコア早見表

### P_Valence (快感度)
```
90-100% → 最高に楽しい！ 🎉
70-89%  → 嬉しい、良い 😊
50-69%  → まあまあ 😐
30-49%  → つらい 😟
0-29%   → 最悪 😢
```

### A_Strength (覚醒度)
```
70-100% → 興奮している！ 🤩
50-69%  → 普通 😐
30-49%  → 落ち着いている 😌
0-29%   → リラックス 😴
```

### D_Extent (支配性)
```
70-100% → コントロール中 💪
50-69%  → 中立的 😐
0-49%   → 受動的 😰
```

### C_Certainty (確信度)
```
70-100% → 確信している ✅
50-69%  → 中程度の確信 😐
0-49%   → 不確実 ❓
```
