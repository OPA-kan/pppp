**→ 🎉 興奮・喜び / ポジティブ**
- 「最高！」「やった！」「大成功！」
- 興奮、幸福感、達成感
- 例：試験合格の報告、恋人とのデート後

---

### 2. 高 P & 低 A (0.6以上 & 0.6未満)
**→ 😊 満足・リラックス / ポジティブ**
- 「良かった」「落ち着いた」「安心」
- 充足感、リラックス、平穏
- 例：好きなことをしている時、緊張が解けた時

---

### 3. 低 P & 高 A (0.6未満 & 0.6以上)
**→ 😰 不安・ストレス / ネガティブ**
- 「やばい」「心配」「ドキドキ」
- 不安、焦燥感、緊張
- 例：プレゼン前、試験中、期限が迫った時

---

### 4. 低 P & 低 A (両方0.6未満)
**→ 😢 悲しみ・落ち込み / ネガティブ**
- 「悔しい」「疲れた」「つらい」
- 抑鬱、疲労感、失望
- 例：失敗したとき、悲しいニュース、疲れきった時

## 📊 詳細なスコア説明

### P_Valence (快感度)

```
0.0 ━━━━━━━━━━━━━━━━━━━━━━ 1.0
↑                        ↑
不快                    快い

0.0 ~ 0.2: 非常に不快
0.2 ~ 0.4: 不快
0.4 ~ 0.5: やや不快
0.5 ~ 0.6: ニュートラル
0.6 ~ 0.8: 快い
0.8 ~ 1.0: 非常に快い
```

**例**:
- 「死にたい」 → 0.05
- 「退屈」 → 0.25
- 「まあ、いいか」 → 0.50
- 「嬉しい」 → 0.75
- 「最高に楽しい！」 → 0.95

---

### A_Strength (覚醒度)

```
0.0 ━━━━━━━━━━━━━━━━━━━━━━ 1.0
↑                        ↑
リラックス              興奮

0.0 ~ 0.2: 非常にリラックス（眠い）
0.2 ~ 0.4: リラックス
0.4 ~ 0.5: やや静か
0.5 ~ 0.6: ニュートラル
0.6 ~ 0.8: アクティブ
0.8 ~ 1.0: 非常に興奮
```

**例**:
- 「眠い...」 → 0.10
- 「落ち着いて」 → 0.35
- 「普通です」 → 0.55
- 「やる気出てきた」 → 0.75
- 「めっちゃ興奮！」 → 0.95

---

### D_Extent (支配性)

```
0.0 ━━━━━━━━━━━━━━━━━━━━━━ 1.0
↑                        ↑
受動的                  支配的

0.0 ~ 0.2: 非常に受動的（コントロール不可）
0.2 ~ 0.4: 受動的
0.4 ~ 0.5: やや受動的
0.5 ~ 0.6: ニュートラル
0.6 ~ 0.8: 支配的
0.8 ~ 1.0: 非常に支配的（完全コントロール）
```

**例**:
- 「どうしようもない」 → 0.15
- 「様子を見守っている」 → 0.40
- 「状況に対応中」 → 0.60
- 「リード している」 → 0.80
- 「完全にコントロール」 → 0.95

---

### C_Certainty (確信度)

```
0.0 ━━━━━━━━━━━━━━━━━━━━━━ 1.0
↑                        ↑
不確実                  確信

0.0 ~ 0.2: 非常に不確実
0.2 ~ 0.4: 不確実
0.4 ~ 0.5: やや曖昧
0.5 ~ 0.6: ニュートラル
0.6 ~ 0.8: 確信している
0.8 ~ 1.0: 非常に確信している
```

**例**:
- 「多分...かな？」 → 0.20
- 「わかりません」 → 0.35
- 「かもしれない」 → 0.55
- 「おそらく正しい」 → 0.75
- 「間違いない！」 → 0.95
