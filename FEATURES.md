# 🌸 Saki PADC 感情分析ダッシュボード - 機能ガイド

## 📋 プロジェクト概要

このダッシュボードは、AI を使用してテキストから複雑な感情を分析し、**PADC (Pleasure-Arousal-Dominance-Certainty)** という4次元モデルで感情を可視化するツールです。

---

## 🎯 PADC スコア解説

### PADC とは？
感情を4つの独立した次元で分析する心理学的モデルです：

#### **P (Pleasantness/Valence) - 快感度**
- **スコア範囲**: 0 ~ 1
- **0に近い** → 不快、嫌な感情 😞
- **1に近い** → 快い、良い感情 😊
- **意味**: そのテキストが表現する感情の肯定性・否定性

#### **A (Arousal/Strength) - 覚醒度**
- **スコア範囲**: 0 ~ 1
- **0に近い** → リラックス、穏やか 😴
- **1に近い** → 興奮、高揚、活動的 🤩
- **意味**: 感情がどれだけアクティブ・ダイナミックか

#### **D (Dominance/Extent) - 支配性**
- **スコア範囲**: 0 ~ 1
- **0に近い** → 受動的、コントロールできない 😰
- **1に近い** → 支配的、コントロールしている 💪
- **意味**: そのテキストが表現する感情的なコントロール感

#### **C (Certainty) - 確信度**
- **スコア範囲**: 0 ~ 1
- **0に近い** → 不確実、曖昧 🤔
- **1に近い** → 確信、明確 ✅
- **意味**: 表現された感情や意見がどれだけ確実か

---

## 🎨 主要機能

### 1. **感情解釈パネル** 🎯
P と A の組み合わせにより、自動的に主要な感情を判定します：

| P > 0.6 | A > 0.6 | 判定 | 絵文字 |
|---------|---------|------|--------|
| ✓ | ✓ | 興奮・喜び | 🎉 |
| ✓ | ✗ | 満足・リラックス | 😊 |
| ✗ | ✓ | 不安・ストレス | 😰 |
| ✗ | ✗ | 悲しみ・落ち込み | 😢 |

**セッションの感情マッピング**:
- **ポジティブ** (P > 0.5): 🌟 肯定的な感情
- **ニュートラル** (0.35 ≤ P ≤ 0.5): 😐 中立的
- **ネガティブ** (P < 0.35): 😔 否定的な感情

---

### 2. **感情スコア分析 (レーダーチャート)** 📊
Recharts ライブラリを使用した**ビジュアルレーダーチャート**：
- 4つのスコア (P, A, D, C) を同時に可視化
- スコアの相対的な大きさが一目で分かります
- 感情の多次元的な特性を直感的に理解可能

---

### 3. **フィードバック統計** 📈
AI の正確性を測定：
- **総分析数**: 累計で分析したテキスト数
- **正解率**: 分析結果が正確だったパーセンテージ
- **合ってる ✅**: 正確な分析の件数
- **違う ❌**: 不正確だった分析の件数

進捗バーでビジュアル的に表示されます。

---

### 4. **分析履歴** 📝
過去の全ての分析結果を表示：
- テキスト内容
- 分析日時
- 各スコア (P, A, D, C)
- フィードバック状態 (✅/❌/❓)

## 📊 使い方ステップガイド

### ステップ1️⃣: テキスト入力
```
感情を分析したいテキストを入力してください。
例）「昨日のプレゼンは成功した！嬉しい！」
```

### ステップ2️⃣: 分析実行
- 「🚀 分析する」ボタンをクリック
- AI が Saki PADC API を呼び出して分析

### ステップ3️⃣: 結果確認
- **感情解釈**: 「🎉 興奮・喜び」などが表示
- **スコア詳細**: P, A, D, C の各値をパーセンテージで表示
- **レーダーチャート**: 4次元の感情をビジュアル表示

### ステップ4️⃣: フィードバック
- 「✅ 合ってる」または「❌ 違う」をクリック
- 統計データが自動更新

### ステップ5️⃣: 履歴確認
- 📝 分析履歴セクションで過去の結果を確認
- 正解率や統計を見てモデルの精度を確認

## 📞 トラブルシューティング

### Q: API エラーが出る場合
**A**: `UPSTASH_REDIS_REST_URL` と `UPSTASH_REDIS_REST_TOKEN` 環境変数を確認してください。

### Q: スコアが 0 に近い場合
**A**: テキストが感情的表現を含まない可能性があります。より感情的な表現を入力してください。

### Q: レーダーチャートが表示されない
**A**: ブラウザのコンソールをチェックして、Recharts ライブラリが正しく読み込まれているか確認してください。
