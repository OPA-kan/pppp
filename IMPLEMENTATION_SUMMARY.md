## 🎭 PADC 感情マッピング

### 感情判定ロジック (P × A)

```
        |  高 A (興奮)        低 A (リラックス)
--------|────────────────────────────────────
高 P (快い) | 🎉 興奮・喜び      😊 満足・リラックス
低 P (不快) | 😰 不安・ストレス    😢 悲しみ・落ち込み
```

### センチメント分類

| P スコア | センチメント | 表示 |
|----------|-----------|------|
| > 0.5 | ポジティブ 🌟 | 緑 |
| 0.35~0.5 | ニュートラル 😐 | 青 |
| < 0.35 | ネガティブ 😔 | 赤 |

### 強度レベル

```
強度値 = √(A² + (P-0.5)² × 2)

0.0~0.5: 低い 🌤️
0.5~0.7: 中程度 ⚡
0.7~1.0: 非常に強い 💥
```
