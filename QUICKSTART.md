# 🚀 クイックスタートガイド

## 📦 セットアップ

```bash
# 依存関係をインストール
npm install

# 環境変数を設定
# .env.local ファイルを作成して以下を追加：
UPSTASH_REDIS_REST_URL=<your-redis-url>
UPSTASH_REDIS_REST_TOKEN=<your-redis-token>

# 開発サーバー起動
npm run dev

# ブラウザで http://localhost:3000 を開く
```

---

## 🎯 基本的な使い方

### 1️⃣ テキスト入力
- テキストエリアに感情的なテキストを入力
- 長めの文章ほど精度が良い

### 2️⃣ 分析実行
- 「🚀 分析する」ボタンをクリック
- 数秒で結果が表示される

### 3️⃣ 結果確認
- **感情アイコン**: メインの感情を表示
- **パーセンテージ**: 各スコアを0~100%で表示
- **レーダーチャート**: 4次元の感情を可視化

### 4️⃣ フィードバック
- 「✅ 合ってる」 or 「❌ 違う」をクリック
- 統計がリアルタイム更新

## 📊 PADC スコアの見方

### P_Valence (快感度) 😊↔️😢
- **高い (70%以上)**: ポジティブ、楽しい、嬉しい
- **中程度 (30~70%)**: ニュートラル、普通
- **低い (30%未満)**: ネガティブ、つらい、悔しい

### A_Strength (覚醒度) 🤩↔️😴
- **高い (70%以上)**: 興奮している、アクティブ
- **中程度 (30~70%)**: 通常の状態
- **低い (30%未満)**: リラックス、落ち着いている

### D_Extent (支配性) 💪↔️😰
- **高い (70%以上)**: コントロール中、支配的
- **中程度 (30~70%)**: 中立的な状態
- **低い (30%未満)**: 受動的、コントロール不可

### C_Certainty (確信度) ✅↔️❓
- **高い (70%以上)**: 確信している、明確
- **中程度 (30~70%)**: ある程度の確信
- **低い (30%未満)**: 不確実、曖昧

## 🎨 ダッシュボードの各パーツ

### 上部: ヘッダー
- ロゴと説明
- 紫→ピンク のグラデーション

### 入力セクション
- テキストエリア: 5行
- 分析ボタン: 紫→ピンク グラデーション

### 結果セクション (分析後に表示)
- **感情表示パネル**
  - 大きな絵文字
  - メイン感情
  - センチメント&強度バッジ
  - 4つのスコアボックス

- **レーダーチャート**
  - 4次元の感情を可視化
  - アニメーション付き

- **フィードバックボタン**
  - ✅ 合ってる (緑)
  - ❌ 違う (赤)

### 統計セクション
- 総分析数、正解数、誤解数
- **正解率をメインで表示**
- プログレスバーと円形表示

### 履歴セクション
- 最新の分析から順に表示
- テキスト、日時、スコア、フィードバック
- 最大50件まで表示可能

## 🔧 トラブルシューティング

### Q: エラー「Saki PADC API failed」
**A**: 環境変数が正しく設定されていることを確認してください。

### Q: スコアが全て0
**A**: API から適切なレスポンスが返されていません。ネットワークを確認。

### Q: レーダーチャートが表示されない
**A**: ブラウザコンソールでエラーを確認。Recharts が正しく読み込まれているか確認。

### Q: 履歴が保存されない
**A**: Redis の接続を確認してください。UPSTASH_REDIS_REST_URL と TOKEN が正しいか確認。
